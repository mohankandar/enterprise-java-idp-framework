# Public Sanitization Notes

This repository was renamed and sanitized for public portfolio/open-source use.

## Applied changes

- Root project renamed to `enterprise-java-idp-framework`.
- Maven group/package changed to `io.github.mohankandar.idp`.
- Module, class, property, and documentation naming changed to public IDP naming.
- Local Git, IDE, build output, and target metadata were removed from the distributable archive.
- Internal repository/distribution management configuration was removed.
- Demo secrets were changed to local placeholders/environment-variable defaults.

## Before publishing

Review any remaining organization-specific examples, URLs, credentials, hostnames, logs, and sample data before pushing to a public remote.
